# UI Guidelines

## Design Philosophy

* Minimal
* Clean
* Professional
* No clutter

---

## Color Palette

* Primary: #1E293B (dark slate)
* Secondary: #64748B (muted gray)
* Accent: #3B82F6 (soft blue)
* Background: #F8FAFC (light neutral)

Avoid:

* overly bright colors
* harsh contrast

---

## Typography

* Font: Inter / system font
* Headings: semi-bold
* Body: regular

---

## Layout

* Max width centered container
* Spacing: consistent (8px grid)
* Cards with soft shadows
* Rounded corners (8px–12px)

---

## Screens

### 1. Dashboard

* “Today’s Opportunities”
* List with:

  * title
  * type
  * urgency badge

---

### 2. Opportunity Detail

* simplified description
* eligibility
* actions:

  * generate receipt
  * play audio

---

### 3. Receipt View

* clean print-style UI
* minimal text
* high readability

---

### 4. Operator View

* list of opportunities
* matched users count

---

## Components

* Card
* Button (primary / secondary)
* Badge (urgent / normal)
* List item

---

## UX Rules

* Max 2–3 actions per screen
* No complex navigation
* Clear labels
* Fast loading
---
## Interaction

* Click → immediate response
* Loading states minimal
* No animations overload

---

## Goal

UI should feel:

* simple to use
* fast
* trustworthy

Not flashy, not complex
